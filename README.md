# 台灣電子股過濾程式

自動掃描台灣電子類股，根據技術指標過濾篩選符合條件的股票。

## 📋 功能特性

### 1. 自動數據抓取
- 自動連接 Google Finance 和 Yahoo Finance 抓取台灣股票市場實時數據
- 支持批量抓取多檔股票數據
- 自動記錄數據抓取時間

### 2. 技術指標分析
- **布林通道（Bollinger Bands）**
  - 20日均線作為中線
  - 上下各 1.23 倍標準差作為上下軌
  - 收盤價突破上軌作為轉強點
- **指數移動平均線（EMA）**
  - 8日均線用於判斷趨勢方向
  - 上升趨勢符合條件

### 3. 多條件過濾
符合以下 **所有條件** 的股票才會被列入報表：

✓ **電子類股**  
✓ **成交量 > 1000張** - 流動性充足  
✓ **收盤價 > 布林上軌** - 技術強勢信號  
✓ **8均線向上** - 中期上升趨勢  
✓ **大戶持倉增加 > 1000張** - 主力進場跡象  

### 4. 自動報表生成
- 生成 Excel 和 CSV 報表
- 報表包含完整的執行日期時間、數據撈取時間、掃描範圍、數據來源
- 詳細的技術指標數據和符合條件說明

### 5. 定時自動執行
- 每個交易日下午 2:00（台灣股市收盤後）自動執行
- 完整的日誌記錄
- 異常處理和錯誤通知

## 🚀 快速開始

### 環境要求
- Python 3.8+
- 活躍的互聯網連接

### 安裝步驟

1. **克隆倉庫**
```bash
git clone https://github.com/ofunasun-collab/stock1.git
cd stock1
```

2. **建立虛擬環境**
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

3. **安裝依賴**
```bash
pip install -r requirements.txt
```

### 使用方式

#### 方式1: 手動執行一次
```bash
python stock_filter.py
```

#### 方式2: 啟動定時排程（推薦）
```bash
python scheduler.py
```

程式會每個交易日下午 2:00 自動執行。

## ⚙️ 配置參數

編輯 `config.py` 可自定義以下參數：

```python
# 布林通道
BOLLINGER_PERIOD = 20        # 均線周期
BOLLINGER_STD_DEV = 1.23     # 標準差倍數

# EMA
EMA_PERIOD = 8               # 8日均線

# 過濾條件
MIN_VOLUME_SHARES = 1000     # 最小成交量（張）
MAJOR_HOLDER_THRESHOLD = 1000 # 大戶持倉增加量（張）

# 執行時間
EXECUTION_TIME = time(14, 0)  # 下午2:00執行
```

## 📋 報表內容

### Excel 報表包含兩個工作表

#### 「摘要」工作表
- 程式執行時間

- 掃描範圍（台灣電子類股）

## 🔧 故障排除

### 問題1: 無法抓取股票數據
**原因**: 互聯網連接問題或 API 限制  
**解決**:
- 檢查互聯網連接
- 檢查 Yahoo Finance API 是否可用
- 稍後重試

### 問題2: 報表為空
**原因**: 沒有股票符合所有條件  
**解決**:
- 檢查市場狀況
- 檢查過濾條件是否過於嚴格
- 查看日誌了解詳情

### 問題3: 定時執行不工作
**原因**: APScheduler 配置問題  
**解決**:
- 確保 APScheduler 已安裝
- 檢查系統時間設置
- 查看日誌文件
